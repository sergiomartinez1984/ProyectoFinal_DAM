Foxy Thief Island es un proyecto creado con Unity

El proyecto se basa en un juego creado con Unity que tiene integrado un flujo de datos que envia el nombre de usuario y su puntuación a una base de datos, en este caso Firebase. Una vez en Firebase y mediante la creación de una aplicación para website esos mismos datos los obtenemos aquí, en la pagina del juego, donde mostramos una tabla de usuarios y sus puntos. pincha Aquí para descargar la APK del juego.

Unity

Unity es un motor de videojuego multiplataforma creado por Unity Technologies. Unity está disponible como plataforma de desarrollo para Microsoft Windows, Mac OS, Linux. La plataforma de desarrollo tiene soporte de compilación con diferentes tipos de plataformas. Unity tiene dos versiones: Unity Professional (pro) y Unity Personal.

Foxy

Nuestro protagonista, Foxy, viene de un conjunto de assets gratuitos que la comunidad de Unity deja libres para su uso, es por ello que se pueden encontrar varios proyectos por internet que usan a este personaje y sus assets como enemigos, decorados, etc..

Firebase

Es una plataforma ubicada en la nube, integrada con Google Cloud Platform, que usa un conjunto de herramientas para la creación y sincronización de proyectos que serán dotados de alta calidad, haciendo posible el crecimiento del número de usuarios y dando resultado también a la obtención de una mayor monetización.
